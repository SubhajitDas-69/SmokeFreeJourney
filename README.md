# SmokeFreeJourney - (Personalized Quit Smoking Assistant)
A full-stack application that delivers motivational and health-based email reminders to help users stay on track with their goals.

## Features
**Personalized Mail Setup** – Users can opt in to receive motivational and health-based messages.
**Daily Email Reminders** – Automatically scheduled personalized emails using `node-cron` and `nodemailer`.
**Smart Stop/Resume** – Users can stop daily mails anytime and re-enable them easily.
**Dynamic Email Templates** – Beautiful EJS-based HTML emails styled for professional readability.
**Secure Authentication** – Using `passport`.

## Tech Stack
**UI :** EJS 
**Backend :** Node.js, Express.js                              
**Database :** MongoDB, Mongoose                                 
**Email Service :** Brevo (SMTP), Nodemailer                                                                      
**Scheduler :** Node-Cron
